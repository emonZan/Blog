# 了解viewport meta tag

## 什么是viewport
viewport, 中文也叫“视口”。在浏览器范畴里，它代表的是在浏览器中，可以看到的网页内容。viewport外的内容在被滚动进来前都是不可见的。

进一步细分viewport，当前可见的部分叫做visual viewport（可视视口）。visual viewport可能会比layout viewport（布局视口）更小，因为当用户缩小浏览器缩放比例时，布局视口不变，而可视视口变小了。
